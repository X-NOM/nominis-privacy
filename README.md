# Nominis GPT Privacy Policy

Official privacy and data usage policy for Nominis Custom GPTs available via the OpenAI GPT Store.

## Overview

This repository hosts the privacy policy for Nominis-operated Custom GPTs, including:
- **AskYT GPT** - Bitcoin-only analytics tool for on-chain activity queries
- **Sanctions Screening GPT** - Compliance-focused blockchain address screening

The policy emphasizes our commitment to:
- Minimal data collection
- No personal data retention
- Transparent data handling practices
- User privacy rights
